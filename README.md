# MJJ - Michael Jackson Tribute Site

Sitio web dedicado al Rey del Pop, Michael Jackson.

## 🚀 Ver Online

**https://myghags.github.io/MJJ/**

> Nota: Si ves un error 404, espera ~2 minutos después de un push para que GitHub Pages se active.

## 🛠️ Tecnologías

- React + Vite
- TypeScript
- CSS

## 📱 Secciones

- **Historia**: Cronología de la vida de Michael Jackson
- **Discos**: Discografía completa (Jackson 5 + Solista)
- **Videos**: Videos musicales por álbum
- **Tracks Random**: 20 canciones aleatorias con reproducción automática
- **Novedades**: Actualidades y novedades

## 💻 Desarrollo Local

```bash
cd mj-king
npm install
npm run dev
```

## 📦 Build para GitHub Pages

```bash
npm run build
```

El build está en `mj-king/dist/` y se deploya desde la rama **main**.
